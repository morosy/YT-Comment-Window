# First Prompt for Codex

あなたは `YT-Comment-Window` プロジェクトの実装担当です。
まず `AGENTS.md` と `PLANS.md` を熟読し、そのルールに従ってください。

## 今回の目的
Chrome 拡張機能の初版を実装してください。

## 初版で必ず実現したいこと
- Manifest V3 の Chrome 拡張機能として成立すること
- 拡張機能アイコンから popup を開けること
- popup に `コメント欄をウィンドウ化` ボタンがあること
- YouTube の通常 watch ページで、そのボタンからコメント欄を右側の独立スクロール領域として表示できること
- 閉じる操作で元に戻せること
- 同一ページで二重起動しても壊れにくいこと

## 技術条件
- Vanilla JavaScript / HTML / CSS のみ
- ビルドツール不要
- 依存ライブラリ不要
- ディレクトリ構成は `AGENTS.md` を基準に作成
- インデントはスペース4つ
- 波括弧は常に付ける

## 実装方針
1. まず最小構成の `manifest.json`、popup、service worker、content script を作成
2. popup から現在タブの YouTube watch ページに開始命令を送る
3. content script で YouTube コメント領域と右カラム挿入先を探索
4. コメント欄を右側の独立スクロールウィンドウへ移す
5. `閉じる` ボタンで復元できるようにする
6. YouTube の SPA 遷移で壊れにくいように最低限のガードを入れる

## 実装時の注意
- YouTube DOM を壊しすぎないこと
- セレクタが見つからない場合は安全に失敗すること
- セレクタやメッセージ種別は共通ファイルへ寄せること
- コメントを複製するより、まずは既存要素の再配置を優先すること
