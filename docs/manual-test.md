# 手動テスト手順

## 事前準備

1. `chrome://extensions/` で unpacked extension として読み込む。
2. デベロッパーツールの Console を開ける状態にしておく。

## 基本確認

1. 通常の YouTube watch ページを開く。
2. popup を開き、`コメント欄をウィンドウ化` を押す。
3. 右カラム上部に `コメント` ヘッダー付きの領域が現れることを確認する。
4. コメント領域だけが独立スクロールすることを確認する。
5. `閉じる` を押し、コメント欄が元の位置へ戻ることを確認する。

## 失敗系確認

1. YouTube トップページや検索結果ページで popup を開く。
2. ボタンが無効化されるか、案内文が出ることを確認する。
3. watch ページでコメント読み込み前に実行し、致命的な console error が出ないことを確認する。

## 再実行確認

1. watch ページで `コメント欄をウィンドウ化` を実行する。
2. もう一度 popup から同じボタンを押す。
3. DOM が二重化せず、壊れないことを確認する。
4. `閉じる` 後に再度実行し、問題なく再配置できることを確認する。

## 遷移確認

1. ウィンドウ化した状態で別の通常動画へ遷移する。
2. 重大な console error が出ないことを確認する。
3. 遷移後のページで再度 popup から起動できることを確認する。

## 画面幅確認

1. ブラウザ幅を広い状態で起動し、右カラム表示を確認する。
2. ブラウザ幅を狭くしてから起動し、安全に起動が拒否されることを確認する。
