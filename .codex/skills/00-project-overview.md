# Skill: Project Overview

## 目的
YouTube の動画ページで、動画を見ながらコメントを閲覧できるようにする Chrome 拡張機能を実装する。

## 初版のゴール
- popup から起動できる
- YouTube watch ページでのみ動く
- コメント欄を右側の独立スクロール領域へ移す
- 閉じると戻せる

## 守ること
- 最小実装を優先
- UI を壊しすぎない
- 依存を増やしすぎない
- 将来拡張しやすい構成にする

## 今回やらないこと
- 設定画面
- 検索機能
- ドラッグ移動
- リサイズ
- Shorts / Live 最適化
