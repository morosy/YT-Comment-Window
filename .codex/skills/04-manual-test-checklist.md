# Skill: Manual Test Checklist

## 基本
- 拡張機能を unpacked load できるか
- popup が開くか
- YouTube watch ページ以外で暴走しないか

## 主要動作
- `コメント欄をウィンドウ化` で起動できるか
- 右側にコメントウィンドウが出るか
- コメント内部だけスクロールするか
- 動画視聴を妨げないか
- `閉じる` で戻せるか

## 境界条件
- コメント読み込み前に押しても壊れないか
- プレイリスト付き動画で極端に崩れないか
- 別動画へ遷移後に再び使えるか
- 同じページで複数回オンオフしても壊れないか

## 確認観点
- console error が出続けていないか
- 元コメント欄の位置復元ができるか
- 右カラム幅が極端に狭い場合の挙動が破綻していないか
